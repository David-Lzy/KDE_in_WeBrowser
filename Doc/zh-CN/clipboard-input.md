# 剪贴板和输入

项目目标是让浏览器里的远程桌面尽量接近本地桌面，尤其是微信和 QQ 的文本复制
粘贴。

## 工作层次

- 浏览器剪贴板权限允许页面在浏览器同意时读写剪贴板。
- Selkies 剪贴板设置负责在浏览器客户端和远程桌面会话之间传递剪贴板内容。
- Wayland/Xwayland 文本剪贴板桥负责让 KDE 原生 Wayland 应用和微信/QQ 这类
  Xwayland 应用之间同步文本剪贴板。

## 预期效果

- 在 Windows/macOS/Linux 复制文本，可以粘贴到浏览器里的 KDE、微信或 QQ。
- 在 KDE、微信或 QQ 复制文本，可以粘贴回本地操作系统。
- 输入法和应用状态集中配置在远程 KDE 桌面里，不需要每台客户端重复搭环境。

## 限制

二进制剪贴板和图片剪贴板取决于 Selkies 与浏览器支持。当前主要保证文本路径。
如果剪贴板不可用，优先检查浏览器站点权限和 Selkies 剪贴板面板。
